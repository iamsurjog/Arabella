# Arabella
## Installation
To download the repo run
### Windows
```bash
git clone https://github.com/iamsurjog/Arabella.git
py -m venv ./.venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

### Linux
```bash
git clone https://github.com/iamsurjog/Arabella.git
py -m venv ./.venv
source .venv/bin/activate
pip install -r requirements.txt
```
## Running it
### Windows
```bash
.\.venv\Scripts\activate
fastapi run main.py
```

### Linux
```bash
source .venv/bin/activate
fastapi run main.py
```
